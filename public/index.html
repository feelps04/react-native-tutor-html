React Native Tutor HTML

Esta é uma página de apresentação para o aplicativo React Native com integração do Google Gemini.

O aplicativo móvel apresenta:

  • Geração dinâmica de perguntas usando a API do Google Gemini
  • Perguntas baseadas em categorias (básico, intermediário, avançado, prático, teórico)
  • Gerenciamento de chave de API para acesso seguro
  • Interface de quiz interativa

Esta versão web é apenas uma página de apresentação. A funcionalidade completa está disponível no aplicativo móvel.
